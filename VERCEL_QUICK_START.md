# ⚡ Vercel 快速部署 - 5 分钟完成

## 🎯 最简单的方法

您的代码已经在 GitHub 上了，现在只需要 5 分钟就能部署到 Vercel！

---

## 📋 快速步骤（5 分钟）

### 第 1 步：访问 Vercel（1 分钟）

打开浏览器，访问：
```
https://vercel.com
```

---

### 第 2 步：登录/注册（1 分钟）

1. 点击右上角 "Log In" 或 "Sign Up"
2. 选择 "Continue with GitHub"
3. 授权 Vercel 访问 GitHub

---

### 第 3 步：导入项目（30 秒）

1. 登录后，点击 "Add New..." → "Project"
2. 找到 `city-matching-test` 仓库
3. 点击 "Import"

---

### 第 4 步：确认配置（30 秒）

Vercel 会自动填充配置，您只需要确认：

- ✅ Framework: Next.js
- ✅ Build Command: `pnpm install && pnpm run build`
- ✅ Output Directory: `.next`
- ✅ Install Command: `pnpm install`

**无需修改任何内容！**

---

### 第 5 步：开始部署（2-3 分钟）

1. 滚动到页面底部
2. 点击 "Deploy" 按钮
3. 等待 2-3 分钟构建完成

---

## 🎉 部署完成！

构建成功后，您会看到：

```
Congratulations!
```

**点击显示的域名链接访问您的应用**：
```
https://city-matching-test.vercel.app
```

---

## ✅ 验证部署

1. 点击域名链接
2. 检查页面是否正常加载
3. 查看 🔒 锁形图标（HTTPS）
4. 测试功能是否正常

---
