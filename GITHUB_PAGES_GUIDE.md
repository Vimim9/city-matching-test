## 📋 部署步骤（2 分钟）

### 第 1 步：进入仓库设置

1. 访问 GitHub 仓库：https://github.com/Vimim9/city-matching-test
2. 点击顶部的 **"Settings"** 标签

### 第 2 步：进入 Pages 设置

1. 在左侧菜单中找到 **"Pages"**
2. 点击进入 Pages 设置页面

### 第 3 步：配置部署

在 "Build and deployment" 部分：

1. **Source**：选择 "Deploy from a branch"
2. **Branch**：
   - 分支：选择 `main`
   - 目录：选择 `/ (root)` 或 `/` 根目录
3. 点击 **"Save"**

### 第 4 步：等待部署

1. 等待 1-2 分钟
2. 页面会显示部署进度
3. 部署完成后，会显示访问链接

### 第 5 步：访问应用

GitHub Pages 会提供一个域名，格式为：
```
https://vimim9.github.io/city-matching-test/
```
